== Description ==

Plugin for developers and designers to:

Remove unnecessary widgets
Remove WordPress bits from admin bar
Add admin notice for client to be careful when editing in admin section
Change log in notification
Remove help
Remove updates from admin bar.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `ag-core.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All done
